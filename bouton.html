Bouton

Usez d'une grosse palette de couleur pour customiser vos boutons.

Pour ce faire, usez de la Class bouton et definissez la couleur du bouton avec bouton-couleur

exemple: bouton-rouge


Bouton
code source
<button class="bouton">Bouton</button>
<button class="bouton-blanc">Bouton</button>
<button class="bouton-gris">Bouton</button>
